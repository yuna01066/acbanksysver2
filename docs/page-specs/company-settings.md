# 회사 설정

## 기본 정보

- Route: `/company-settings`
- 주요 파일: `src/pages/CompanySettingsPage.tsx`, `src/components/company/*`
- 주요 사용자: 관리자, 중간관리자
- 화면 목적: 회사 정보, 견적 기본 문구, 휴일, 권한 정책을 설정한다.

## UI 구조

- 상단: 회사 설정 제목, 탭/저장 액션
- 섹션: 회사 정보, 견적 기본 문구, 휴일, 페이지 권한, 기능 권한
- 입력폼: 회사명, 사업자 정보, 계좌, 연락처, 문구
- 관리 표: 권한/휴일 목록

## 톤앤매너

- 회사 정보는 견적서 출력에 영향을 주므로 안정적인 폼 레이아웃을 유지한다.
- 권한 설정은 표 형태로 명확히 보여준다.
- 저장 성공/실패 메시지를 짧게 제공한다.

## 자주 수정하는 요소

- 견적 기본 문구
- 회사 정보 폼 라벨
- 권한 테이블
- 휴일 관리 카드

## 연결 데이터와 주의점

- 연결 데이터: 회사 설정, 견적 템플릿, `page_role_access`
- 권한: 관리자/중간관리자 전용
- 변경 위험: 회사 정보는 고객용 견적서/PDF에 그대로 노출될 수 있다.
