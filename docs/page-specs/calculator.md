# 견적/수율 계산기

## 기본 정보

- Route: `/calculator`
- 주요 파일: `src/pages/Calculator.tsx`, `src/components/CalculatorWidget.tsx`, `src/components/PanelCalculator.tsx`
- 관련 로직 문서: `docs/logic-specs/quote-calculation-logic.md`, `docs/logic-specs/yield-calculation-logic.md`
- 주요 사용자: 견적 작성 직원, 관리자
- 화면 목적: 원판 수율과 판재/제품 견적을 계산하고 초안 또는 발행 견적 흐름으로 연결한다.

## UI 구조

- 상단: 페이지 제목, 초안함, 관리자 설정 액션
- 견적 모드: 초안 탭/저장 상태, 소재/컬러/두께/사이즈/가공 선택, 계산 결과
- 수율 모드: 원판/조각 입력, 네스팅 결과, 잔재 판단
- 결과 영역: 총액, 계산 근거, 검수 필요/불가 사유

## 톤앤매너

- 입력 단계가 많기 때문에 장식보다 단계 구분과 현재 선택 상태가 중요하다.
- 선택 버튼은 크기와 정렬을 맞추고, 비활성/불가 상태를 명확히 표시한다.
- 계산 근거는 직원이 고객에게 설명할 수 있는 문장으로 유지한다.

## 자주 수정하는 요소

- 가공 카테고리/옵션 노출 순서
- 계산 결과 breakdown 문구
- 초안 툴바 정렬과 저장 상태 표시
- 생산 불가/검수 필요 메시지

## 연결 데이터와 주의점

- 연결 데이터: 원판/색상/사이즈/옵션 단가, `quote_drafts`, 견적 계산 유틸
- 권한: 견적 작성은 로그인 사용자, 관리자 설정 이동은 권한 체크 필요
- 변경 위험: UI 문구 변경처럼 보여도 계산 로직, 단가 snapshot, 초안 자동 저장과 연결될 수 있다.

## 정보 위계

- 1순위: 현재 계산 모드, 선택된 원판/가공 조건, 최종 견적가
- 2순위: 계산 근거, 생산 불가/검수 필요 사유, 초안 저장 상태
- 3순위: 관리자 설정, 초안함 이동, 보조 옵션
- 낮은 강조: 장식성 배경, 설명성 긴 문구

## 상세 설계 기준

- 단계 구성: 소재/컬러/두께/사이즈/가공/결과 순서를 유지하고, 사용하지 않는 소재 버튼은 노출하지 않는다.
- 선택 UI: 선택 버튼은 같은 행에서 높이와 정렬을 맞추고 긴 옵션명은 줄바꿈을 허용한다.
- 접착가공: 원판 기준 무기포 접착은 단순 조건 중심으로 보여주고 박스 길이 측정처럼 복잡한 입력은 제품 제작 흐름으로 분리한다.
- 결과 표시: 최종 금액 옆에는 `계산 가능`, `검수 필요`, `생산 불가` 상태가 함께 보여야 한다.
- 모바일: 옵션 그리드는 2열 이하로 줄이고 계산 결과는 접히지 않게 상단 또는 하단 고정 영역을 피한다.

## 검수 시나리오

- `type=quote`에서 새 초안 생성, 옵션 선택, 계산 결과 확인, 초안 저장 상태를 확인한다.
- `type=yield`에서 원판/조각 입력 후 결과가 화면 폭을 넘지 않는지 확인한다.
- 단가 미등록 조합은 최종가 0원처럼 보이지 않고 명확한 차단 사유를 보여준다.
- 접착가공 선택 시 불필요한 입력이 나타나지 않고 계산 근거가 중복되지 않는다.
