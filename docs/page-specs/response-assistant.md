# 상담 응대 보조

## 기본 정보

- Route: `/response-assistant`
- 주요 파일: `src/pages/ResponseAssistantPage.tsx`
- 주요 사용자: 상담 직원, 관리자
- 화면 목적: 고객 문의에 대한 내부 응대 문안과 근거를 빠르게 작성한다.

## UI 구조

- 상단: 로고/홈 이동, 검색 또는 입력 영역, 관리 화면 이동
- 본문: 문의 입력, 추천 답변, 근거, 복사 액션
- 보조: 관련 태그, 참고 문서, 빈 상태
- 오류 상태: 분석 실패 또는 근거 없음 안내

## 톤앤매너

- 상담원이 빠르게 읽고 복사할 수 있게 문장과 버튼을 간결하게 유지한다.
- 추천 답변은 내부 참고용이며 최종 발송은 사람이 확인한다는 톤을 유지한다.
- 입력 영역과 결과 영역의 시각적 위계를 명확히 둔다.

## 자주 수정하는 요소

- 입력 placeholder
- 추천 답변 카드
- 복사 버튼과 성공 메시지
- 근거 없음 안내

## 연결 데이터와 주의점

- 연결 데이터: 응대 지침/근거 데이터, AI Gateway 또는 Edge Function
- 권한: 로그인 사용자 중심, 관리 기능은 관리자 권한
- 변경 위험: 고객에게 직접 발송하는 흐름으로 바꾸지 않는다.
