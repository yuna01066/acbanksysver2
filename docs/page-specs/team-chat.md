# 팀 채팅

## 기본 정보

- Route: `/team-chat`
- 주요 파일: `src/pages/TeamChatPage.tsx`, `src/components/TeamChatCard.tsx`
- 주요 사용자: 전체 직원
- 화면 목적: 사내 메시지, 업무 공유, 멘션, 일정/견적 관련 알림성 대화를 확인한다.

## UI 구조

- 상단: 팀 채팅 제목, 채널/검색/필터
- 메시지 영역: 메시지, 작성자, 시간, 멘션/태그
- 입력 영역: 텍스트 입력, 첨부/전송 액션
- 보조: 알림/읽음/빈 상태

## 톤앤매너

- 채팅 말풍선은 유지하되 대시보드에서는 색감을 낮춘다.
- 메시지 본문과 메타 정보의 위계를 명확히 둔다.
- 업무 태그는 눈에 띄되 전체 화면을 지배하지 않게 한다.

## 자주 수정하는 요소

- 말풍선 색상과 간격
- 멘션/태그 표시
- 입력창 높이와 버튼 배열
- 빈 채팅 안내

## 연결 데이터와 주의점

- 연결 데이터: 채팅 메시지, 사용자 프로필, 멘션 제안
- 권한: 로그인 사용자
- 변경 위험: 실시간 구독/메시지 전송 로직을 UI 리팩터링과 함께 건드리지 않는다.

## 정보 위계

- 1순위: 최신 메시지, 작성자, 입력창
- 2순위: 멘션/태그, 첨부/링크, 읽음/시간
- 3순위: 채널/필터, 검색 결과
- 낮은 강조: 장식성 말풍선 효과

## 상세 설계 기준

- 메시지: 본문은 읽기 쉬운 줄간격을 유지하고 작성자/시간은 보조 텍스트로 낮춘다.
- 말풍선: 본인/타인 메시지 구분은 색상 차이를 낮게 유지하고 과한 채도는 피한다.
- 입력: 전송 버튼은 항상 같은 위치에 두고 멘션 자동완성은 입력창을 가리지 않게 한다.
- 대시보드 카드: 홈의 팀 채팅 카드는 전체 채팅보다 더 compact하고 색감을 낮춘다.
- 모바일: 입력창은 키보드가 올라와도 전송 버튼이 보이게 한다.

## 검수 시나리오

- 긴 메시지, URL, 해시태그가 말풍선 밖으로 넘치지 않는다.
- 멘션 입력 시 제안 목록이 화면 밖으로 잘리지 않는다.
- 새 메시지 도착 시 스크롤 위치 정책이 의도와 맞는지 확인한다.
- 홈 카드의 최근 메시지와 전체 채팅의 메시지가 같은 순서로 보인다.
