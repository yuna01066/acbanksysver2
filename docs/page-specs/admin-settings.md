# 관리자 설정

## 기본 정보

- Route: `/admin-settings`
- 주요 파일: `src/pages/AdminSettingsPage.tsx`
- 주요 사용자: 관리자, 중간관리자
- 화면 목적: 가격, 회사, 권한, 템플릿, 연동 설정 등 관리 기능으로 진입하는 허브다.

## UI 구조

- 상단: 관리자 설정 제목, 홈/뒤로 액션
- 섹션: 가격/원판, 회사/문서, 직원/권한, 외부 연동, 시스템 관리
- 카드: 기능명, 설명, 접근 권한, 이동 버튼
- 제한 상태: 권한이 없을 때 안내

## 톤앤매너

- 설정 허브는 메뉴판처럼 단순하고 예측 가능해야 한다.
- 카드 제목, 아이콘, 설명 길이를 통일한다.
- 위험한 설정은 일반 설정보다 시각적으로 구분한다.

## 자주 수정하는 요소

- 설정 카드 그룹
- 관리자 전용 배지
- 외부 연동 설명
- 접근 제한 문구

## 연결 데이터와 주의점

- 연결 데이터: 페이지 접근 권한, 회사 설정, 각 관리 페이지
- 권한: 관리자/중간관리자 중심
- 변경 위험: 메뉴 노출은 보안 경계가 아니므로 실제 권한 guard와 RLS를 유지해야 한다.
