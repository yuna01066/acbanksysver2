# 원판/색상/사이즈 관리

## 기본 정보

- Route: `/panel-management`
- 주요 파일: `src/pages/PanelManagementPage.tsx`, `src/components/panel-management/*`
- 주요 사용자: 관리자, 중간관리자
- 화면 목적: 판재 제품, 컬러, 사이즈, 두께, 추가금과 생산 가능 조합을 관리한다.

## UI 구조

- 상단: 원판 관리 제목, 탭/관리 액션
- 관리 영역: 제품 선택, 컬러 관리, 사이즈 관리, 옵션/추가금 관리
- 목록/폼: 생산 가능 여부, 단가, 조색비, 양단면 추가금
- 피드백: 저장 성공/오류, 단가 미등록 안내

## 톤앤매너

- 관리자 화면이므로 장식보다 입력 정확도와 검증 메시지를 우선한다.
- 컬러/사이즈/두께 조합은 표나 그룹으로 정리해 실수 가능성을 낮춘다.
- 거래처명은 외부 노출 화면에 나오지 않게 관리용 정보로만 둔다.

## 자주 수정하는 요소

- 컬러 속성/배지
- 사이즈별 단가 입력 표
- 생산 불가 조합 표시
- 양단면/사틴/아스텔/조색비 문구

## 연결 데이터와 주의점

- 연결 데이터: `panel_masters`, `panel_sizes`, `panel_option_surcharges`, `panel_pricing_versions`
- 권한: 관리자/중간관리자 전용
- 변경 위험: 단가 변경은 기존 저장 견적에 자동 반영되지 않아야 한다.

## 수정 체크리스트

- [ ] 생산 불가 조합이 계산기에서 차단되는가?
- [ ] 단가표 표시명에 거래처명이 노출되지 않는가?
- [ ] 변경 전후 기존 견적 snapshot이 유지되는가?
- [ ] 입력 표가 모바일에서 사용할 수 있는 최소 폭을 확보하는가?
