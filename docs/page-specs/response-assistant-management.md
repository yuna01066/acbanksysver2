# 상담 응대 관리

## 기본 정보

- Route: `/response-assistant-management`
- 주요 파일: `src/pages/ResponseAssistantManagementPage.tsx`
- 주요 사용자: 관리자, 중간관리자
- 화면 목적: 상담 응대 보조에 쓰이는 지침, 근거, 템플릿을 관리한다.

## UI 구조

- 상단: 관리 제목, 새 지침/저장 액션
- 목록: 지침/템플릿 제목, 카테고리, 상태, 수정일
- 편집: 제목, 본문, 태그, 사용 여부
- 보조: 미리보기, 검증 메시지

## 톤앤매너

- 지침 편집 화면은 문서 편집기처럼 차분하고 넓게 유지한다.
- 활성/비활성 상태를 명확히 보여준다.
- 긴 템플릿은 접힘 또는 미리보기로 관리한다.

## 자주 수정하는 요소

- 카테고리/태그 표시
- 편집 폼 라벨
- 미리보기 영역
- 저장/비활성 액션

## 연결 데이터와 주의점

- 연결 데이터: 응대 지침/템플릿 테이블, AI 응대 생성 흐름
- 권한: 관리자/중간관리자 전용
- 변경 위험: 잘못된 지침은 상담 답변 품질에 직접 영향을 준다.

- [ ] 저장 전 미리보기로 문안 확인이 가능한가?
- [ ] 삭제보다 비활성 처리를 우선하는가?
- [ ] 일반 직원이 관리 화면에 접근할 수 없는가?
