# 견적 초안함

## 기본 정보

- Route: `/quote-drafts`
- 주요 파일: `src/pages/QuoteDraftsPage.tsx`, `src/services/quoteDrafts.ts`, `src/contexts/QuoteContext.tsx`
- 주요 사용자: 견적 작성 직원
- 화면 목적: 여러 견적 초안을 저장, 전환, 복제, 보관하고 필요한 초안을 발행한다.

## UI 구조

- 상단: 초안함 제목, 새 초안/계산기 이동 액션
- 목록: 초안명, 수정일, 수신자/합계/상태, 열기/복제/보관/발행 액션
- 선택 발행: 체크박스와 발행 결과 피드백
- 빈 상태: 초안이 없을 때 새 초안 생성 유도

## 톤앤매너

- 브라우저 탭처럼 빠르게 전환한다는 느낌을 유지한다.
- 초안명과 마지막 수정 시간이 먼저 읽혀야 한다.
- 발행 견적서 목록과 혼동되지 않도록 초안 상태 배지를 분명히 둔다.

## 자주 수정하는 요소

- 초안 카드/테이블 레이아웃
- 저장 상태/발행 상태 배지
- 선택 발행 결과 메시지
- 빈 상태 안내

## 연결 데이터와 주의점

- 연결 데이터: `quote_drafts`, `saved_quotes`
- 권한: 작성자 본인 초안만 조회/수정/삭제
- 변경 위험: 초안은 실제 견적번호를 확정하지 않고 발행 시점에 확정한다.
